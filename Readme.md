# VoiceAIWrapper

A simple full-stack project combining a modern Vite + Tailwind frontend with a Django backend.

## Tech Stack
- **Frontend:** Vite, Tailwind CSS  
- **Backend:** Django

## Setup References
- Frontend:  
  - https://vite.dev/guide/  
  - https://tailwindcss.com/docs/installation/using-vite
- Backend:  
  - https://docs.djangoproject.com/en/6.0/intro/tutorial01/

## Getting Started
1. Setup the frontend using Vite and Tailwind.
2. Initialize the backend with Django.
3. Run both servers locally and connect FE to BE as needed.
